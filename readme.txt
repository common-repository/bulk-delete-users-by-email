== Description ==

Delete users by entering plaintext which will be searched for email addresses and matching user accounts then deleted.

== Installation ==

1. Install the pugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for "Delete Users" in your WordPress admin section sidebar
